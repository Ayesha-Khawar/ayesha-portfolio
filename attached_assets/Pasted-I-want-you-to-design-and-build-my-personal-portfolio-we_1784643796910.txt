Since this website must deploy on Vercel or Netlify, do NOT use local vector databases or self-hosted RAG pipelines.

Instead, implement the assistant using one of these free, deployment-friendly approaches:

The chatbot must never hallucinate. If the answer isn't in my portfolio, it should politely say it doesn't have that information.
